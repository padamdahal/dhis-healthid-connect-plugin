# DHIS2 FHIR Tracker Capture Plugin

Simple DHIS2 Tracker Capture form field plugin.

## Features
- Select identifier type (from DataStore)
- Enter ID value
- Search FHIR server
- Auto-fill TEI attributes

## Setup
1. Upload to GitHub
2. Open in Codespaces
3. Run build (if integrated into DHIS2 App Platform)

## Config (DataStore)
/api/dataStore/healthid-connect/config
